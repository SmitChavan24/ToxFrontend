This is your customer's website

This page simulates what your customer's website would look like with the HelpWave chat widget installed. Click the chat bubble in the bottom-right corner to try it out!

📋 Embed Code

<!-- Add this before </body> on your website -->
<script 
    src="https://your-helpwave-domain.com/widget.js"
    data-helpwave-url="https://your-helpwave-domain.com"
></script>

Optional attributes: data-position="left" to place on the left side, data-color="#10b981" to change the bubble color.

🚀 Zero Config

Just paste one <script> tag. No framework dependencies, no build steps, works everywhere.

📱 Mobile Ready

Widget goes fullscreen on mobile for the best chat experience on small screens.

🎨 Customizable

Change position, colors, and more with simple data attributes. Matches any brand.
